# Pixel Color Randomizer 
<sup>or RE for short</sup>

I like pixel art. I like creative works. I'm not creative. What's the solution?

## What is RE?

RE is a python script that takes an image as an input and outputs a batch of pixelated, colorshifted versions.

Colorshifting occurs by preforming mathematical operations with random variables on each RGB channel of each pixel.
